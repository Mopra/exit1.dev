Exit1 Video Components

Pure HTML/CSS components extracted from the Exit1 dashboard for use in marketing videos

Check Table

Main data table showing all checks with status, URL, response time, and actions. Includes sorting and column controls.

Check Form - Step 1

Check type selection: Website, API Endpoint, TCP Port, or UDP Port.

Check Form - Step 2

Basic information: URL, display name, check frequency, immediate re-check, down confirmation.

Check Form - Step 3

Advanced configuration: HTTP method, request headers, response validation, cache control.

Status Badges

Status indicators including Online, Offline, Redirect, Error, Paused states. Also includes SSL and type badges.

Check Card (Mobile)

Card-based layout for checks, used on mobile devices. Compact view with all essential information.

Check Map View

Global infrastructure map with animated ping flows, data markers, status legend, and callout cards showing endpoint details.

Bulk Actions Bar

Fixed bottom bar for bulk operations when multiple checks are selected. Enable, disable, or delete multiple checks.

Page Header

Page title, description, and action buttons. Multiple variants with search, tabs, and usage counter.

Domain Intelligence Header

Domain Intelligence page header with "Enable for checks" button for adding domains.

Domain Intelligence Stats

Overview stat cards showing total, expiring soon, healthy, and error counts for monitored domains.

Domain Intelligence Table

Full table of monitored domains with status badges, expiry dates, registrar info, and actions menu.

Enable Domain Intelligence Modal

Modal dialog for selecting which checks to enable Domain Intelligence monitoring for.

Domain Intelligence Settings

Side panel showing detailed domain information, alert thresholds, registration info, and nameservers.

Domain Intelligence Alert

Alert notification showing domain expiring soon with warning message and action buttons.

How to Use

These components are designed for use in video editing software that supports HTML/CSS rendering. Each component is self-contained and uses the shared styles.css for base styling.

  • All components use pure HTML and CSS - no JavaScript required
  • Dark theme optimized with glassmorphism effects
  • Icons are inline SVGs for easy customization
  • Demo data is pre-filled - modify as needed
  • Fonts used: DM Sans (body) and Space Mono (monospace)
  • All colors use CSS custom properties for easy theming
  • See DOMAIN-INTELLIGENCE-README.md for video storyboard suggestions
